# Установка сервера в Docker
1. Установить Docker
2. Клонировать настоящий репозиторий
3. Выполнить команду:
```shell
docker build .
```
4. Подожать окончания билда
5. Выполнить команду
```shell
docker-compose up
```
6. Все последующие запуски осуществлять через Docker container.
## Ссылка на сервер в Docker http://localhost:8080
